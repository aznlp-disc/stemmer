# Azerbaijani Language Stemmer

Stemmer for Azerbaijani language written in Python.

## Usage
```
root  
└── stemmer  
└── yourmain.py  
└── ...
```  
```python
from stemmer import Stemmer

stemmer = Stemmer()
lst = stemmer.stem_words(["nümunə",])
```
